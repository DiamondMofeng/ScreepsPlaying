## npm scripts  

### development

Rollup source code then copy them into `./default`   
```bash
npm run dev
```


Rollup source code then copy them into the default folder of local server    
```bash
npm run dev:local
```


Use jest to test.
```bash
npm run jest
```


Check lint problems
```bash
npm run lint
```


Fix automatically fixable lint problems. (Most times useless tbh)
```bash
npm run lint:fix
```
